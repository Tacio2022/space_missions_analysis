# space_missions_analysis
The main goal of this project is analyze how the "space race" scenario changed over the years, from 1957 to 2020.

## The data source is from:  nextspaceflight.com 

## Requirements:

- Python 3.9
- Pandas
- Numpy
- Matplotlib
- Seaborn
- Iso3166
- Plotly
- Panel
- HVplot.pandas
